# SweatStake 

**Stake SOL on your gym goals. Hit them, win. Miss them — the pot goes to those who showed up.**

Skin in the game meets fitness accountability. Built on Solana.

![SweatStake Banner](https://via.placeholder.com/800x400?text=SweatStake+Banner) <!-- Replace with real screenshot later -->

## About

SweatStake is a Web3 fitness accountability app where users stake real SOL on personal gym/fitness goals. 

- **Hit your goal** → Claim your share of the pot.
- **Miss your goal** → Your stake fuels the winners.

No more empty promises. Real consequences. Real results.

## Key Features

- Create custom gym/fitness goals with SOL stakes
- Strict camera-only verification (no gallery photos)
- Live proof requirements + timestamping
- Community & private staking pools
- Solana wallet integration (Phantom, Solflare, etc.)
- Transparent on-chain payouts

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Base44 AI Platform
- **Blockchain**: Solana
- **Auth**: Email + Wallet
- **Storage**: IPFS / Arweave (for proof photos)
